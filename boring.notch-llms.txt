# Boring Notch

> Boring Notch is a macOS app that transforms the MacBook Pro notch into a dynamic, interactive control center. Inspired by iOS Dynamic Island, it provides music playback controls with live album art and visualizer, calendar and reminders integration, webcam mirror, file shelf with AirDrop/quick share, battery/charging status, system HUD replacements (volume, brightness, backlight), download monitoring, onboarding, and extensive customization — all built with SwiftUI, AppKit, and the Battery Activity Monitor framework.

## Architecture

### App Entry Point

- **`boringNotchApp.swift`** — `@main` SwiftUI `App` struct. Declares the `MenuBarExtra` menu and owns the `AppDelegate`. Initializes Sparkle (`SPUStandardUpdaterController`) for auto-updates and passes it to `SettingsWindowController`.

### Delegate & Window Management

- **`AppDelegate`** (`boringNotchApp.swift`) — `NSApplicationDelegate`. Creates, positions, and manages one `BoringNotchSkyLightWindow` per display. Handles screen configuration changes, screen lock/unlock visibility, drag detection, window alpha for display switching, onboarding, and the "Show on All Displays" multi-screen behavior.

### Key Patterns

- **Multi-screen:** When `Defaults[.showOnAllDisplays]` is `true`, `AppDelegate` creates one `BoringViewModel` + `NSWindow` per `NSScreen`. Otherwise a single `vm` / `window` pair serves the selected display.
- **Notch Space:** `NotchSpaceManager` creates a dedicated `CGSSpace` at level `Int32.max` to float above everything (including full-screen apps).
- **SkyLight:** `BoringNotchSkyLightWindow` optionally enables SkyLight (used to render above the lock screen when `Defaults[.showOnLockScreen]` is enabled).
- **Media Control:** `MusicManager` is a facade over pluggable controllers (`MediaControllerProtocol`). Supports Now Playing, Apple Music, Spotify, YouTube Music. Auto-falls back from `NowPlayingController` to `MPRemoteCommandCenter` on macOS 15.4+.
- **HUD Replacement:** When `Defaults[.hudReplacement]` is enabled, `MediaKeyInterceptor` (via XCPHelper) intercepts system volume/brightness/backlight key events and routes them through Boring Notch's own overlays instead of the system HUD.

## Window Sizing & Geometry

From `sizing/matters.swift`:

```swift
let openNotchSize = CGSize(width: 640, height: 190)
let windowSize = CGSize(width: openNotchSize.width, height: openNotchSize.height + shadowPadding) // +20 for shadow
let shadowPadding: CGFloat = 20
let cornerRadiusInsets = (
    opened: (top: 19.0, bottom: 24.0),
    closed: (top: 6.0, bottom: 14.0)
)
let MusicPlayerImageSizes.size = (
    opened: CGSize(width: 90, height: 90),
    closed: CGSize(width: 20, height: 20)
)

func getClosedNotchSize(screenUUID: String?) -> CGSize  // Computes width from safeAreaInsets
func getScreenFrame(_ screenUUID: String?) -> CGRect?
```

Closed notch height:
- Notch screen + `matchRealNotchSize` → `screen.safeAreaInsets.top`
- Notch screen + `matchMenuBar` → menubar height
- Notch screen + `custom` → `Defaults[.notchHeight]`
- Non-notch screen + `matchMenuBar` → menubar height
- Non-notch screen + `custom` → `Defaults[.nonNotchHeight]`

Closed notch width: `screen.frame.width - auxiliaryTopLeftArea.width - auxiliaryTopRightArea.width + 4`

## Key Technical SkyLight Details

The app uses private `SkyLight.framework` APIs via `dlopen`/`dlsym`:

- `SLSCreateSpace`, `SLSAddWindowsToSpaces`, `SLSRemoveWindowsFromSpaces`, `SLSSetWindowLevel` etc.
- Used to create a dedicated "notch space" at level `Int32.max` so the notch floats above everything.
- `BoringNotchSkyLightWindow.enableSkyLight()` / `disableSkyLight()` toggle visibility on the lock screen.
- `NotchSpaceManager` encapsulates the `CGSSpace` handle.

## Important Implementation Notes

1. **No Notch on Non-Notch Macs:** The app uses a fallback floating style (`Style.floating`) for Macs without a physical notch.
2. **Bypass Gatekeeper:** First launch from DMG requires running `xattr -dr com.apple.quarantine`.
3. **First Launch:** Triggers onboarding + welcome sound (`boring.m4a`).
4. **Multi-Screen:** `showOnAllDisplays` creates independent `BoringViewModel` + `NSWindow` per screen, indexed by `displayUUID`.
5. **Media Sources:** `MusicManager` auto-detects if Now Playing is deprecated (macOS 15.4+) and falls back to `MPRemoteCommandCenter`.
6. **Accessibility Authorization:** HUD replacement requires accessibility permissions via the XPC helper.
7. **Drag Detection:** File drags over the notch region auto-open the shelf (`expandedDragDetection`).
8. **Lyrics:** Fetched synced (LRC) or unsynced lyrics lines matched to `elapsedTime`.
